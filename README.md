# Bluetooth-Manager-for-dArkOS-
# R36S Bluetooth Manager – dArkOs Edition  A complete Bluetooth manager for R36S running dArkOs.   Supports enable/disable, scan, pair, connect, disconnect and forget devices.   Auto-installs dependencies, fixes PulseAudio &amp; RetroArch audio, and ensures stable A2DP connections.   Multilingual interface with dialog-based UI.
